Deleting Task

Keyword: nil


Command Format

Example

delete {task number}

delete 1


Note:


Delete command can execute while filtering tasks/searching for tasks.



When the task is deleted successfully, the program will stay on the page where the task was deleted. In this scenario, task 'something' was to be deleted.
