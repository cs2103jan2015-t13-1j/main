Filter Tasks

Keyword: nil


Command

Filtered Tasks

view all

All tasks will be displayed

view complete

All completed tasks will be displayed

view incomplete

All incomplete tasks will be displayed

view today

All tasks beginning or due on that day itself will be displayed

view tomorrow

All tasks beginning or due on the next day itself will be displayed

view {high/medium/low}

All tasks with requested rank will be displayed

view {deadline/timed/floating}

All tasks with requested task type will be displayed

view overdue

All tasks which are overdue will be displayed


Note:


In this scenario, user wants to view overdue tasks.
