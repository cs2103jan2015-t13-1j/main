Loading Task(s) list from File

Keyword: nil


Commands available:


Command Format

Example

Remarks

load {filename}

load mylist.txt

File will be loaded and program will work on this file thereafter


Note:


Load command can be requested anytime.


Once another file is loaded, it will be used as the file for user to work on the next visit.


If user close the program without the proper command 'exit' or save/saveas command through out the entire program, the program will open default file 'storage.txt' for user to work on.
