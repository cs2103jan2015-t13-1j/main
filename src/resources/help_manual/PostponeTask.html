Postpone Task

Keyword: by


Commands available:


Command Format

Example

Remarks

postpone {task number} by {number of days} {day/days}

postpone 1 by 2 days

Only applicable for non-deadline type task

postpone {task number} by {number of hours} {hour/hours}

postpone 1 by 2 hours

Only applicable for non-deadline type task


Note:


Postpone command can execute while filtering tasks/searching for tasks.


In this scenario, the user wants to postpone the task 'go on vegetarian diet' by 1 hour.
